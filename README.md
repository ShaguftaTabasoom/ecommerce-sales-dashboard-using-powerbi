# E-Commerce Sales Dashboard (Power BI)

This project presents an interactive E-Commerce Sales Dashboard built using Power BI, designed to analyze and monitor key sales metrics, customer behavior, and product performance. The dashboard provides a clean and insightful representation of various KPIs to support data-driven decision-making.

# Project Overview

The E-Commerce Sales Dashboard aims to deliver a comprehensive view of business performance across multiple dimensions such as sales, profit, quantity, customers, categories, states, and payment modes.
It helps stakeholders quickly identify trends, understand product performance, and evaluate regional sales distribution.

# Key Features & Insights
### 1. KPI Cards

Four KPIs are highlighted at the top:

Total Sales

Total Profit

Total Quantity Sold

Average Order Value (AOV)

These metrics give an at-a-glance understanding of business performance.

### 2. State-wise Sales Analysis

A bar chart visualizing Sum of Amount by State, showcasing which regions contribute the most to revenue.

### 3. Category-wise Quantity Distribution

A donut chart representing Quantity by Category (e.g., Clothing, Electronics, Furniture).

### 4. Monthly Profit Trend

A bar chart tracking Profit Month-wise, helping identify profitable and loss-making months.

### 5. Customer-wise Sales

A horizontal bar chart showing Top Customer Sales, highlighting high-value customers.

### 6. Payment Mode Distribution

A donut chart showing Quantity by Payment Mode such as COD, UPI, Debit Card, EMI, etc.

### 7. Sub-Category Wise Profit

A bar chart representing Profit by Sub-Category (Printers, Saree, Bookcases, etc.).
